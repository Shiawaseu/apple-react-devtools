## About
This is an **unofficial port of the React Devtools** extension for Safari on AppleOS.

## Features

- Anything `React-Devtools` can do!
- Works natively in Safari on all Apple devices

---

## Installation
- You have the option to either:
  - Install the extension directly via the [Releases](https://github.com/Shiawaseu/apple-react-devtools/releases) tab
  - Clone and compile yourself
<details>
<summary>Installation</summary>

### 1. Prerequisites

- You already have a built application ready, either by the `releases` tab or you have built one yourself.
- ![](/assets/1.png)
### 2. Launching the application (macOS)
- Since the application is not signed, you will receive a warning regarding security, you can ignore it in your settings (`System Settings > Privacy & Security`)

![](/assets/2.png)

### 3. Enable the extension
- You might need to enable the `Allow Unsigned Extensions` setting under **Safari extension settings**
![](/assets/3.png)
</details>
